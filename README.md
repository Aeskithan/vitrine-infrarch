# Site vitrine INFRARCH

Site vitrine statique de la société **INFRARCH** — bureau d'étude en ingénierie réseau, sécurité & automatisation.

- `index.html` — page unique du site
- `css/style.css` — feuille de style (charte INFRARCH : bleu `#007ebd`, gris `#5f5f5f`, police Futura BT Extra Black)
- `js/main.js` — menu mobile + formulaire de contact (démo, aucun backend branché)
- `assets/` — logos, favicon et police

Hébergé sur GitHub Pages.
